# Social Preview and Brand

The repository social preview source lives in the main repo:

- `docs/assets/social-preview.svg`
- `docs/assets/social-preview.png`

## How to Set the GitHub Social Preview

1. Open the Suka.dev repository on GitHub.
2. Go to **Settings** -> **General** -> **Social preview**.
3. Upload `docs/assets/social-preview.png`.
