# Roadmap

The active roadmap is tracked in GitHub Issues.

## Current Project Tasks

- #1 Build live agent presence publishing for CLI and coding agents.
- #2 Add WebSocket realtime updates for dashboard state.
- #3 Build claim and conflict workflow in the dashboard.
- #4 Create shared decision memory for agents and teams.
- #5 Add `.suka` project configuration for domains, privacy, and modes.
- #6 Design hosted and self-hosted team workspace foundation.
- #7 Improve repo intelligence with imports, package relationships, routes, and tests.
- #8 Build team connection UI for inviting and connecting collaborators.
- #9 Harden production quality: logging, CI, Docker, security, and release workflow.

## Product Direction

Suka should become the coordination layer for professional AI-assisted software teams.

The dashboard should feel like a serious workspace: a repo map, mission control surface, conflict radar, and team connection hub.
