# Dashboard

The dashboard is the visual command center for Suka.

## Current Views

- **Agents sidebar**: shows active agent presence.
- **Repo Domain Map**: visualizes repository areas and relationships.
- **Inspector**: shows details for the selected repo area or agent.
- **Radar**: surfaces claims, failures, and risk signals.
- **Activity**: shows recent coordination events.

## Canvas Controls

- Select.
- Zoom in/out.
- Fit graph.
- Local focus.
- Risk focus.
- Hide/show sidebars.

When a sidebar is open, use its header icon to hide it. When hidden, it fully leaves the layout and a single canvas-edge icon appears to restore it.

## Responsive Behavior

The dashboard is designed for:

- Desktop browsers.
- Tablet browsers.
- iOS Safari-sized operational views.

Visual QA checks:

- No desktop/mobile overflow.
- Repo map renders.
- Inspector selection works.
- Sidebars hide and restore.
- Local storage state persists.
