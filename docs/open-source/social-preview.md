# Social Preview

The source template for Suka.dev's repository social card lives at:

- `docs/assets/social-preview.svg`
- `docs/assets/social-preview.png`

Recommended GitHub usage:

1. Open the repository on GitHub.
2. Go to **Settings** -> **General** -> **Social preview**.
3. Upload `docs/assets/social-preview.png`.

Design intent:

- Make `Suka.dev` readable at small card sizes.
- Show the product category immediately: realtime coordination for AI coding agents.
- Hint at the dashboard's core mental model: agents, claims, repo map, and conflict-aware edges.
- Keep the source editable in git so the preview can evolve with the product.
